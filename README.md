# Highliight Saver

Highlight, organize, and never lose a thought. Save text from any website, customize highlight colors in real-time, and add personal notes. Manage and search all your saved insights from one central dashboard.


# Key Features:
- Instant Highlighting: Simply select any text on any webpage to save it instantly.
- Dynamic Colors: Change highlight colors directly on the page to categorize your thoughts visually (e.g., Yellow for facts, Green for quotes).
- Add Context: Attach personal notes to any highlight to capture your "aha!" moments.
- Stay Organized: Pin your most important snippets to the top of your collection for quick reference.
- Powerful Search: Find exactly what you need in seconds with a fast, intuitive search interface in the extension home.
- Centralized Dashboard: Access all your highlights, notes, and source links from a single, clean workspace.

# Why Choose Highlight Saver?
- Lightweight & Fast: Designed to stay out of your way until you need it.
- Privacy Focused: Your data stays with you, no unnecessary tracking.
- Productivity First: Built for power users who need to bridge the gap between reading and remembering.


# Technical Overview & Permissions:
- storage: Used to save highlights, notes, and user preferences locally.
- activeTab / host_permissions: Necessary to inject the highlighting CSS and scripts onto the pages the user is actively reading.
- contextMenus: Enables the "Save Highlight" option in the right-click menu.
- Data Collection: This extension does not collect or transmit any personal data or browsing history to external servers. All highlights are stored locally within the browser's storage API.
